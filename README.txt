Webdriver - Java Demo:

Pasos previos:
- Instalar SDK Java 8+ y variables de entorno (JAVA_HOME + Path)
- Instalar Maven y variables de entorno (MVN_HOME + M2_HOME)
- Instalar Git

Intrucciones de ejecucion:
Parado en la carpeta del proyecto, clonar el repositorio de GitHub:
Cmnds:
git init
git clone https://github.com/Julian0182/webDriver_Java.git

Ubicarse en la carpeta que contiene el POM.xml y correr los siguientes comandos mvn:
mvn clean compile test
